# موقع المنيو الرقمي للكافيه - دليل الاستخدام

## نظرة عامة

موقع منيو رقمي احترافي باللغة العربية بالكامل، مصمم خصيصاً لعرض مشروبات الكافيه من خلال QR Code. الموقع للعرض فقط ولا يحتوي على نظام طلبات أو دفع إلكتروني.

### 2. تسجيل الدخول

بعد إنشاء الحساب، يمكنك تسجيل الدخول من خلال:
- الذهاب إلى `/login`
- إدخال البريد الإلكتروني وكلمة المرور
- النقر على "تسجيل الدخول"

### 3. إدارة المنتجات

بعد تسجيل الدخول، ستتمكن من:

- ✅ تعديل الأسعار

## الصفحات الرئيسية

### 1. الصفحة الرئيسية (/)
- عرض جميع المشروبات المتاحة
- البحث عن المشروبات
- التصفية حسب القسم (ساخن/بارد)
- عرض الأسعار بالجنيه المصري

### 2. صفحة تسجيل الدخول (/login)
- تسجيل دخول المدير
- حماية لوحة التحكم

### 3. لوحة التحكم (/dashboard)
- إحصائيات المنتجات
- إدارة المنتجات (CRUD)
- رفع الصور
- معلومات QR Code

---

## إنشاء QR Code

لإنشاء رمز QR للمنيو:

1. انسخ رابط الموقع من لوحة التحكم
2. استخدم أحد المواقع التالية لإنشاء QR Code:
   - https://www.qr-code-generator.com/
   - https://www.qrcode-monkey.com/
   - https://goqr.me/
3. اطبع الرمز وضعه على الطاولات

## التقنيات المستخدمة

- **Frontend**: React + TypeScript
- **Routing**: React Router
- **Styling**: Tailwind CSS v4 + خط Cairo
- **Icons**: Lucide React
- **Backend**: Supabase (Database + Auth + Storage)
